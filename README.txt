FRESH TOMATO MOVIES
===================
This application launches a static page showing some interesting films.
If you click the poster images you will be shown a trailer for the film.

SETUP
=====
1. Requires Python3

2. At the project root, run the command 'python3 entertainment_centre.py'

ADDING FILMS
============
To add films for display, add entries to the movies.json file, entering the following:

title
storyline
poster_image_url
trailer_youtube_url
duration
rating
